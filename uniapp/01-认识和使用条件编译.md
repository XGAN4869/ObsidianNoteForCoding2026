# 认识和使用条件编译

## 什么是条件编译

条件编译是 uni-app 提供的一种机制，允许在不同平台环境下编译不同的代码，实现**多端差异处理**。

官方文档：https://uniapp.dcloud.net.cn/tutorial/platform.html

---

## 语法格式

条件编译使用 **预处理指令**，以 `#` 开头，放在 JavaScript 注释中：

```javascript
// #ifndef VUE3
// 如果不是 Vue3 环境，编译此代码块（Vue 2 分支）
// #endif

// #ifdef VUE3
// 如果是 Vue3 环境，编译此代码块（Vue 3 分支）
// #endif
```

### 指令说明

| 指令 | 含义 |
|------|------|
| `#ifndef xxx` | 如果**不定义** xxx，则编译 |
| `#ifdef xxx` | 如果**定义**了 xxx，则编译 |
| `#if xxx` | 如果 xxx 为真，则编译 |
| `#endif` | 条件结束标识 |

---

## 实际项目应用

本项目 `main.js` 中使用条件编译同时兼容 Vue 2 和 Vue 3：

```javascript
// #ifndef VUE3
import Vue from 'vue'
import './uni.promisify.adaptor'
Vue.config.productionTip = false
App.mpType = 'app'
const app = new Vue({
  ...App
})
app.$mount()
// #endif

// #ifdef VUE3
import { createSSRApp } from 'vue'
export function createApp() {
  const app = createSSRApp(App)
  return {
    app
  }
}
// #endif
```

### 编译结果

- 在 **Vue 2 环境**编译：只保留 `#ifndef VUE3` 到 `#endif` 之间的代码
- 在 **Vue 3 环境**编译：只保留 `#ifdef VUE3` 到 `#endif` 之间的代码

| 变量 | 含义 |
|------|------|
| `VUE3` | Vue 3 环境 |
| `H5` | H5 网页环境 |
| `MP-WEIXIN` | 微信小程序 |
| `APP-PLUS` | App 环境 |

---

## 注意事项

1. 条件编译指令不是普通 JavaScript 注释，不会被浏览器执行
2. `#ifndef` = "if not defined"，`#ifdef` = "if defined"
3. 必须成对使用，有开头就有 `#endif`
